### Method 1: Direct File Opening
1. Download or clone this repository to your local machine
2. Navigate to the project folder
3. Double-click on `resumepage.html` to open it in your default web browser

### Method 2: Using a Local Server (Recommended)

#### Option A: Using Python
If you have Python installed:

**Python 3:**
```bash
python -m http.server 8000
```

**Python 2:**
```bash
python -m SimpleHTTPServer 8000
```

Then open your browser and visit: `http://localhost:8000/resumepage.html`

#### Option B: Using VS Code Live Server
1. Install the "Live Server" extension in Visual Studio Code
2. Right-click on `resumepage.html`
3. Select "Open with Live Server"

#### Option C: Using Node.js http-server
```bash
npx http-server
```

Then open your browser and visit the URL provided in the terminal.
